# Fillo - Weblab GraphQL api

[![js-standard-style](https://cdn.rawgit.com/feross/standard/master/badge.svg)](http://standardjs.com)

## Development

### ENV variables

```
$ cp .env.sample .env
```

### Run server

```
$ npm start
```

### Tests

```
$ npm test
```

#### Linter

We're using ESLint with Standard config

```
$ npm run lint
```
